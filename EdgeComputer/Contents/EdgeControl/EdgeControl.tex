\chapter{\textcolor{red}{Arduino Edge Control}}

\URL{https://docs.arduino.cc/tutorials/edge-control/getting-started-edge-control/}

Um den Arduino Edge Control in der Arduino IDE zu konfigurieren, folge diesen Schritten:

Lade die Arduino IDE herunter, wenn du sie noch nicht installiert hast, auf der offiziellen Website von Arduino (arduino.cc).

Starte die Arduino IDE und wähle \menu{Tools} im Hauptmenü.

Klicke auf ``Board'', und wähle ``Arduino Edge Control'' in der Dropdown-Liste.

Gehe weiter zu \menu{Port}, wähle den korrekten Port der seriellen Schnittstelle, an dem dein Arduino Edge Control angeschlossen ist.

Klicke auf \menu{Browse} (Durchsuchen) in der ``Boards Manager URL''-Eingabe.

Gibt es neue Versionen des Arduino Edge Control-Zusatzmoduls, dann würdest du ``Install'' (Installieren) für die gewünschte Version klicken.

Schließe den Fenster ``Boards Manager'' und gehe zurück zum Menü \menu{Tools}.

Wähle \menu{Manage Libraries} (Bibliotheken verwalten), um die erforderlichen Bibliotheken für die Arduino Edge Control-Kommunikation zu installieren.

Suchen und Installiere die Bibliotheken ``NewSoftPwm'' und ``Arduino\_Edge\_Control'', falls sie noch nicht installiert sind.

Schließe den Fenster ``Library Manager''.

Lade einen geeigneten Arduino Edge Control-Code für dein Projekt und überprüfe, dass alle erforderlichen Bibliotheken verwendet werden.

Klicke auf den Pfeil unten im Menü \menu{Upload} (Hochladen) und lade den Code auf dein Arduino Edge Control.

Mit diesen Schritten ist die Arduino Edge Control in der Arduino IDE konfiguriert.

\section{ARD PRO AKX00034 Arduino Pro Edge Steuerung, nRF52840}

Arduino Edge Control
Für den Außenbereich optimierte Lösung zur Fernüberwachung und -steuerung,
Sammeln Sie Echtzeitdaten von intelligenten Sensoren und nutzen Sie diese direkt vor Ort.

Die Steuerung kann überall positioniert werden und eignet sich für Precision Farming, Smart Agriculture und andere Anwendungen, die eine intelligente Steuerung an abgelegenen Orten erfordern. Die Stromversorgung kann entweder über ein Solarpanel oder einen DC-Eingang erfolgen.

Die Arduino Edge Control ist in der Lage, Sensoren anzuschließen und Aktoren wie Verriegelungsventile anzutreiben. Darüber hinaus kann der gesamte Prozess in Echtzeit überwacht werden und dadurch produktionsbedingte Risiken minimieren.

\bigskip

Besondere Merkmale

\begin{itemize}
  \item Bluetooth BLE onboard
  \item erweiterbar durch alle MKR-Boards - 2G/3G/CatM1/NBIoT, LoRa, Sigfox and WiFi/BT
  \item steuerbar über Arduino Cloud (Fernzugriff)
  \item Echtzeituhr mit CR2032 Backup Batterie
  \item Solarmodul-Batterieladegerät
\end{itemize}

\bigskip

Technische Daten
\begin{itemize}
  \item Microcontroller: nRF52840 (64 MHz ARM® Cortex-M4F)
  \item Ladecontroller: LT3652
  \item 6x digitale Eingänge
  \item 16x digitale Ausgänge
  \item 8x analoge Eingänge ( 0 - 5 V )
  \item 4x analoge Eingänge ( 4 - 20 mA )
  \item 16x analoge Sensoreingänge
  \item 4x Halbleiterrelais ( 60 V / 2,5 A)
  \item 16x Relaisausgänge
  \item Schnittstellen: QSPI/SPI/TWI/I²S/PDM/QDEC
  \item 2 Steckplätze für MKR-Boards
  \item 12 V Spannungsversorgung
  \item Maße: 104 x 86 mm
\end{itemize}

\bigskip

Die Ein- und Ausgänge des Boards sind für den Einsatzzweck optimiert. So sind die sechs digitalen Eingänge als flankengesteuerte Aufwachpins ausgelegt und 16 Sensoreingänge stehen explizit für hydrostatische Wasserstandsmarken zur Verfügung. Des weiteren sind 8 der 16 Relaisausgänge mit zusätzlichen Treibern versehen.

Durch die Möglichkeit einen Bleiakku anzuschließen und diesen über den integrierten Laderegler zu laden, erhalten Sie eine großartige Unabhängigkeit. Die Batterielaufzeit beträgt dabei dank des unglaublich sparsamen Boards bis zu 34 Monate bei einer 5 Ah Batterie.

\bigskip

Anwendungen
\begin{itemize}
  \item smarte Gewächshäuser
  \item Hydro- / Aquakulturen
  \item Champignonzucht
\end{itemize}

\section{ARD EDGE CE KIT Arduino Pro - Edge-Control Gehäuse-Kit}


ARD EDGE CE KIT
Arduino Edge Control Enclosure Kit

Das Arduino Edge Control Enclosure Kit wurde für Anwendungen in der Industrie und der intelligenten Landwirtschaft entwickelt und ist der perfekte Begleiter für den Arduino Edge Control.
Es bietet dem Modul ein robustes Gehäuse, das es vor den Elementen, Staub und versehentlichen Stößen schützt. Es ist IP40-zertifiziert und kompatibel mit DIN-Schienen, wodurch es sicher und einfach in jedes Standard-Rack oder jeden Schrank passt.

Darüber hinaus verfügt das Kit über ein zweizeiliges, 16-stelliges LC-Display mit weißer Hintergrundbeleuchtung und einen programmierbaren Druckknopf, so dass es vom Benutzer angepasst werden kann, um Sensordaten wie Wetterbedingungen und Bodenparameter sofort zu visualisieren. Bei jedem Tastendruck können verschiedene Daten an Ort und Stelle und in Echtzeit angezeigt werden, ohne dass eine Konnektivität erforderlich ist.

\bigskip

Spezifikationen

\begin{itemize}
  \item robustes und kompaktes Schutzgehäuse für den Außen-/Industrieeinsatz
  \item einfache Installation und Organisation in Racks oder Schränken
  \item IP40-zertifizierter Schutz
  \item LC-Display zur sofortigen Überprüfung der Sensordaten vor Ort
  \item anpassbare Drucktaste zur Anzeige verschiedener Daten im Wechsel
  \item überwachen Sie Daten, auch wenn eine Verbindung nicht verfügbar oder unzuverlässig ist
\end{itemize}

\bigskip
