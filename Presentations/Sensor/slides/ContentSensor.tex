\Mysection{Sensoren und Aktoren}

\STANDARD{Sensoren}
{ 
	
  \framesubtitle{Einleitung}	
  Sensor (lat: \glqq Fühler" ) dienen zur \textbf{qualitativen} und \textbf{quantitaiven} Auffassung von physikalischen, chemischen, biologischen, klimatischen und medizinischen Größen. \cite{Hering:2023} \\ 
  Jedes Sensorsystem besteht aus folgenden Komponenten: 
  \begin{itemize}
  	\item \textbf{Sensor-Element}
  	\item \textbf{Auswert-Elektronik}
  \end{itemize}
}
